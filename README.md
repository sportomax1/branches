# Hearth & Bean — Dev site

This branch contains assets and a simple static site for the Hearth & Bean coffee shop concept.

- `coffee_shop.html` — main landing page
- `menu.html` — sample menu
- `contact.html` — contact form (mailto)
- `styles.css` — site styles
- `coffee_shop_business_model.md` — business model and projections
- `financials.csv` — simple monthly financial template

Open `coffee_shop.html` in a browser to preview the site locally.
